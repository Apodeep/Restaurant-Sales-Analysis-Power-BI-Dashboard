# 🏆 Restaurant Sales Intelligence Dashboard
**Author:** Abdulrhman Apodeep  
**Role:** Junior Data Analyst  

## 📌 Project Overview
This project is a full Business Intelligence solution built using Power BI to analyze restaurant sales performance, customer behavior, and revenue trends. The dashboard transforms raw transactional data into interactive visual insights that support data-driven decision-making.

## 🎯 Business Objectives
- Monitor Total Revenue & Profitability  
- Analyze Sales by Category & Product  
- Identify Peak Sales Periods  
- Understand Customer Purchasing Patterns  
- Track KPIs with dynamic filtering  

## 🛠 Tools & Technologies
- Power BI  
- Power Query (Data Cleaning & Transformation)  
- DAX (Measures, KPIs, Time Intelligence)  
- Data Modeling  
- Excel / CSV Dataset  

## 📊 Key Features
- Interactive Filters & Slicers  
- Drill-through & Drill-down Analysis  
- KPI Cards  
- Trend Analysis  
- Dynamic Date Filtering  

## 📈 Key Insights
- Identified top-performing product categories  
- Discovered peak revenue months  
- Highlighted underperforming products  
- Revealed customer purchasing behavior trends  

## 🚀 Business Impact
This dashboard enables business managers to:
- Optimize inventory planning  
- Improve marketing strategies  
- Increase operational efficiency  
- Make data-driven decisions
